flex-basis 속성

아이템의 기본크기 설정 width와 flex-basis 모두 요소의 너비를 설정하는 속성이지만 flex-basis는 flex 컨테이너 안에서만 동작한다. 요소의 크기가 변동되기 전의 기준값 역할을 한다.
flex 레이아웃 내에서는 flex-basis가 width보다 우선순위가 높다.
